<objective>
Fix four device-mismatch crash sites on GPU/MPS backends, document device compatibility conventions, add a lint guard script, and create parametrized device smoke tests.

Purpose: SciPy lfilter crashes on MPS tensors, AutoTCL loss functions build labels on CPU then index device tensors, and contrastive losses do unnecessary numpy round-trips for indexing. MPS is the strictest backend — if code runs on MPS it runs everywhere. The fix follows one rule: tensors created inside forward passes or losses must inherit device from input tensors.

Output: Fixed crash sites, device convention documentation, lint guard, smoke tests.
</objective>
