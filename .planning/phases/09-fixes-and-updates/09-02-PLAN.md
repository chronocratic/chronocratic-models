<objective>
Collapse BasicEncodingMixin from 4 hooks to 2 hooks and add gradient_enabled to both mixin families (D-02, D-03).

Purpose: The 4-hook contract (_get_encoder + _get_encoder_module + _prepare_inputs + _postprocess) is broken because TST and Series2Vec return bound methods from _get_encoder(), breaking the auto-detect in _get_encoder_module(). The 2-hook contract (_get_encoder -> nn.Module, _encode_batch) is honest: module for state toggle, callable for compute. gradient_enabled allows backprop through encoder for adversarial attacks.
Output: Refactored BasicEncodingMixin, updated dilated BaseEncodingMixin, 7 model updates, deleted old hooks, gradient_enabled working on both families.
</objective>
